# Editorial Brief Workspace

This repository contains a seeded editorial workspace used by the live test matrix.

## Layout

- `briefs/source-brief.md`
  - working memo from the research role
- `briefs/publication-brief.md`
  - final editor-owned brief
- `notes/audience-signals.md`
  - supporting audience and market notes

## Verification

Run `./scripts/verify.sh` after substantive changes.
