# Agirunner Python SDK

Minimal Python client for Agirunner platform APIs.

This package covers the workflow and project surfaces added during the dashboard parity wave:

- auth token exchange
- task and workflow listing/loading
- project listing/loading
- project timeline
- resolved workflow config
- workflow documents
- task artifacts
- planning workflow creation
- phase gate and phase cancel actions

The client uses the Python standard library by default and supports transport injection for tests.
